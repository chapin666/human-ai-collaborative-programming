## 本书简介

**目标读者**：有2-5年经验、正在使用或准备使用AI IDE的程序员  
**核心承诺**：读完能立即改变工作方式，提升30%+效率  
**篇幅**：26章，约550页，33万字  
**风格**：60%实践 + 40%理论

> "AI 不会取代程序员，但会用 AI 的程序员会取代不会用的。"
